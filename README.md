# Todo List App

## Description
A simple Todo List application using Node.js, Express, SQLite, and a frontend built with HTML, CSS, and JavaScript.

## Setup Instructions

1. Navigate to the backend directory:
   cd backend

2. Install dependencies:
   npm install

3. Initialize the database:
   node database.js

4. Start the server:
   npm start

5. Open the frontend:
   Open frontend/index.html in a web browser.
