# Todo List App

## Description
A simple Todo List application using Node.js, Express, SQLite, and a frontend built with HTML, CSS, and JavaScript. Includes automated testing with Jest and Supertest.

## Setup Instructions

1. Install dependencies (from root directory):
   npm install

2. Start the server (from root directory):
   npm start

3. Open the frontend:
   Open frontend/index.html in a web browser.

## Testing

- Run tests (from root directory):
  npm test

- Run tests in watch mode:
  npm run test:watch

Tests include unit tests for API endpoints and integration tests for data persistence in SQLite.
